Regular Expressions

Lookahead and lookbehind

Reference: http://www.regular-expressions.info/lookaround.html

There is a pattern to it. Positive is ‘=’ and negative is ‘!’. Lookbehind is prefixed with ‘<’.

Lookahead: (?=#)

Negative Lookahead: (?!#)

Lookbehind: (?<=#)

Negative Lookbehind: (?<!#)
